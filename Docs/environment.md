# Development Environment

## Project

GeoAI Flood Risk Decision Agent

## Operating System

Windows 11

## Python Version

Python 3.12.10

## Virtual Environment

Project-specific virtual environment (.venv)

## Environment Strategy

The project uses an isolated Python virtual environment to ensure reproducibility and prevent dependency conflicts.

## Installation Method

Virtual environment created using:

py -3.12 -m venv .venv

Activated using:

.venv\Scripts\Activate.ps1

## Notes

All project dependencies will be recorded in requirements.txt.
