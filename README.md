# GeoAI Flood Risk Decision Agent

## Overview

A GeoAI-powered spatial decision support system for assessing flood risk in Nairobi County, Kenya.

## Objectives

- Integrate raster and vector datasets
- Compute flood risk indicators
- Generate explainable risk scores
- Support decision-making using GeoAI

## Technology Stack

- Python
- GeoPandas
- Rasterio
- QGIS
- Streamlit
- GitHub
